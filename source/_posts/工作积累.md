## 计算函数执行时间
```csharp
private static void CalculateExcutedTime(string tag, Action func)
{
    var oldTime = DateTime.Now;
    func();
    var newTime = DateTime.Now;
    Log("Execute time of \"" + tag + "\" is " + (newTime - oldTime).TotalSeconds + " seconds");
}
```
